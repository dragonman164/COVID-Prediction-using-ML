Introduction

This is a simple web app that predicts whether the person being tested is Covid Positve or negative. This Web App works through a Machine Learning Model which runs on the backend to predict the results. These results include basic symtomps like fever, headache, cough , difficulty in breathing etc. It is strictly adviced not to rely on the results of this web application because this application is still in testing phase
